# 🚀 Final Project - Portfolio Web Application

## 📌 Task 5: Final Project and Optimization

As part of my internship final task, I built a fully functional, responsive, and performance-optimized portfolio website using only HTML, CSS, and JavaScript.

---

## 💡 Features

- ✅ Responsive Layout (mobile & desktop)
- 🌙 Dark/Light Mode Toggle with LocalStorage
- 🖼️ Lazy Loaded Images for faster load time
- ⚡ Minified CSS & JavaScript
- 🌍 Cross-Browser Compatibility
- 💬 Contact Form (Mock Submission)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM Manipulation, LocalStorage)
